## 📖 Descripción del Proyecto
Este repositorio contiene el software interactivo `practica4TeoCom.py`, desarrollado con la librería **Flet**. El programa es un entorno integral de Teoría de la Computación que permite explorar las expresiones regulares, simular Autómatas Finitos Deterministas (AFD) y No Deterministas (AFND), y convertir un AFD a su Expresión Regular equivalente utilizando el algoritmo de eliminación de estados.
